== Description ==

Extend WordPress REST API to support remote site management, plugin updates, and integration with FluentCRM, FluentSupport, and other popular plugins.

== Installation ==

1. Upload the `jrb-remote-site-api-openclaw` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure your API token in the 'JRB Remote API' settings page.
